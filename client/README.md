# Backend & DevOps Portfolio

A modern, interactive portfolio template designed for Backend and DevOps engineers. Features a sleek, terminal-inspired aesthetic with interactive "window" previews for showcasing projects.

## Features

- **Interactive Tech Grid**: Click on technology cards (JS, Python, Docker, etc.) to open detailed views.
- **Window Modal System**: Mimics an OS window interface for project details.
- **Project Previews**: Sections for video previews of backend systems.
- **Documentation Journey**: Timeline view of your learning path.
- **Responsive Design**: Works on desktop and mobile.
- **Animation**: Smooth CSS transitions and glassmorphism effects.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (no external CSS frameworks)
- **Icons**: External URLs (easily replaceable with local assets)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization

-   **Content**: Edit `src/data.js` to update your projects, skills, and journey.
-   **Styling**: Modify `src/App.css` to change colors, fonts, or layout.
-   **Images/Videos**: Replace the placeholder URLs in `src/data.js` with your own assets.

## Aesthetic

The design uses a dark, high-contrast theme suitable for developers, with neon accents corresponding to each technology's brand color.
